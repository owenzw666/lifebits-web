
# Lifebits Web – Location-Based Life Logging System

## 📌 Overview

Lifebits Web is a cloud-based web application that allows users to create, manage, and visualize location-based personal notes.

This project is designed to demonstrate full-stack development skills using modern technologies and cloud deployment practices.

---

## 🚀 Features

* User authentication (Register / Login)
* Create, edit, and delete personal notes
* Location-based note visualization on an interactive map
* Responsive web interface
* Cloud deployment on Azure

---

## 🛠️ Tech Stack

### Frontend

* React
* Axios
* (Optional) MapLibre / Leaflet for map integration

### Backend

* ASP.NET Core Web API
* Entity Framework Core

### Cloud & Database

* Azure App Service
* Azure SQL Database

---

## 📍 Project Goals

* Build a real-world full-stack web application
* Demonstrate RESTful API design
* Implement cloud deployment using Azure
* Showcase integration of geospatial (map-based) features
