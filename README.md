# 🌍 Lifebits - Map-based Note Taking App

## 📌 Project Overview | 项目简介

**English:**
Lifebits is a full-stack map-based note-taking application that allows users to create, view, edit, and delete notes directly on a map.  
It demonstrates modern web development practices including React frontend, .NET backend, RESTful APIs, and interactive map integration.

**中文：**
Lifebits 是一个基于地图的全栈记事本应用，用户可以在地图上创建、查看、编辑和删除笔记。  
该项目展示了现代 Web 开发实践，包括 React 前端、.NET 后端、RESTful API 以及地图交互功能。

---

## 🚀 Features | 功能

- 📍 Create notes by clicking on the map  
  点击地图创建笔记

- 🗺️ Display notes as markers on the map  
  在地图上显示笔记标记

- 💬 Popup view for note details  
  点击标记显示笔记详情

- ✏️ Edit notes directly inside popup  
  在 popup 中直接编辑笔记

- 🗑️ Delete notes from popup or list  
  支持从 popup 或列表删除笔记

- 📋 Note list with improved UI (card style, hover effects)  
  左侧列表展示（卡片式 UI + hover 效果）

- 🔄 List and map synchronization  
  列表与地图联动（定位 + 打开 popup）

- 🎯 Custom marker styling  
  自定义 marker 样式（替代默认图标）
